Exemplo de Formulário


a tag "select" cria uma lista que é possível selecionar.

existe nesse código um LABEL atributo e um elemento, segue exemplo:
elemento: <label for="iest">Estado: </label>
atributo: <optgroup label="Região Sudeste"></optgroup>. neste caso ele está colocando um "título" e "separando" a lista dentro da separação do comando "optgroup"

existe nesse código um SELECT atributo e um elemento, segue exemplo:
elemento: <select name="estado" id="iest"></select>
atributo: <option value="" selected>--- Escolha ---</option>. neste caso ele está "substituindo" a tag "value", mas existe a diferença que o "value" vai pré preencher e o "select" faz outra coisa, vai estudar.

o comando "optgroup" serve para, neste caso, separar a lista que o select criou

a lista Profissão, diferente da lista de Estado, tem a possibilidade de escrever e não só selecionar e a diferença é que tem o comando "input"

Comparativo Rápido
Característica select datalist
Entrada de dados Apenas clique/seleção. Digitação + Sugestão.
Flexibilidade Rígida (Siga a lista). Flexível (Livre, com dicas).
Interface Abre um menu drop-down fixo. Filtra as opções conforme você digita.
Agrupamento Suporta <optgroup> (como você usou). Não suporta agrupamentos visuais (labels).
