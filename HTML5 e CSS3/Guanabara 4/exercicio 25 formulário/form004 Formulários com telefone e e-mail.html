Exemplo de formulário

Dados Pessoais

a tag "fieldset" coloca uma "borda" em volta do formulário; é possível configurar pelo css3

a tag "legend" serve como o título do formulário, funciona de uma forma parecida com a tag "caption" das tabelas.

a tag "pattern" serve para inserir Regex como no caso do telefone, segue exemplo: pattern="^\(\d{2}\)\d{4,5}-\d{4}$"

a tag "value" substitui os valores da tag "placeholder"

required: serve para bloquear a tela até que os dados sejam preenchidos.

minlength="5": serve para determinar o mínimo de caracteres.

maxlength="30": serve para determinar o máximo de caracteres.

a tag "type" define o tipo de bloco.
size="10": serve para determinar quantos caracteres vai aparecer no bloco.

placeholder="Nome do Usuário": serve para marcar dentro do bloco o que deve ser inserido, serve mais como um auxílio.

autocomplete="username" ou autocomplete="current-password"...: colocar o autocomplete="" na tag "form" é diferente de colocar na tag "input" e aparentemente funcionam independente.

autocomplete="off" serve para o seu navegador não mostrar opções quando for preencher o formulário. method="post" evita que as informações apareçam na URL.

Em todo formulário deve ter a tag LABEL onde o "for" vai se referir ao "id" que está no "input" respectivo. A tag <label> no HTML5 serve para definir rótulos (legendas) para elementos de formulário (<input>, <textarea>, etc.)
