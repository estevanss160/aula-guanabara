Principais formatações:

Negrito / destaque

Nesta frase tem um termo em Negrito usando a tag/comando:
<b>texto em negrito</b> (não semântica)

Nesta frase tem um termo em destaque usando a tag/comando:
<strong>texto em destaque</strong> (semântica)

Itálico / ênfase:

Nesta frase tem um termo em itálico usando a tag/comando:
<i>texto em itálico</i> (não semântica)

Nesta frase tem um termo em ênfase usando a tag/comando:
<em>texto em ênfase</em> (semântica)

Texto marcado:

Posso criar também um texto marcado usando a tag/comando:
<mark>texto marcado</mark> os textos são naturalmente marcados com amarelo

Posso criar também um texto marcado com outra cor usando a tag/comando:
<mark style="background-color: nome da cor em inglês">texto a ser marcado</mark>

Texto grande e pequeno:

Estamos criando um texto grande e um pequeno neste parágrafo usando respectivamente as tag's/comandos:
<big>texto grande</big>
a palavra "big" está obsoleta então é melhor formatar o tamanho do texto no CSS
<small>texto pequeno</small>

Texto deletado:

posso marcar um texto como excluído para indicar que ele pode ser lido, mas não considerado. Usando a tag/comando:
<del>texto para "excluir"</del>

Texto inserido(eu chamo de sublinhado):

posso marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usando a tag/comando:
<ins>texto para ser inserido</ins>

Texto sobrescrito:

para inserir coisas do tipo potência x2030+3 ou mais simples x20+3 usei a tag/comando:
x<sup>20<sup>30</sup></sup>+3 ou mais simples x<sup>20</sup>+3

para subescrever

para inserir coisa do tipo H2O usei a tag/comando:
H<sub>2</sub>O
