Esta é a segunda página do meu site

gostaria de retornar a primeira página? se sim então terá duas opções: a primeira você retornará nesta mesma aba e a segunda retornará criando outra aba.
Estou usando o comando <a href="colocar o link aqui">palavra ou frase para o link</a>
segue exemplo:

A primeira opção: <a href="index.html">clique aqui</a>
A primeira opção: clique aqui.

na segunda opção usarei o seguinte comando:
<a href="colocar o link aqui" target="_blank" rel="prev">palavra ou frase para o link</a>,
segue exemplo:

A segunda opção: <a href="index.html" target="_blank" rel="prev">clique aqui</a>
A segunda opção: clique aqui
