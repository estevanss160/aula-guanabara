Terceira página

aqui usarei o seguinte comando para retornar para a primeira página ou para a segunda:
<a href="../nome da página ou link">palavra ou frase para o link</a>
segue exemplo:
Voltar para a primeira página: <a href="../index.html">clique aqui</a>
Voltar para a primeira página: clique aqui

Voltar para a segunda página: <a href="../pagina2.html">clique aqui</a>
Voltar para a segunda página: clique aqui
