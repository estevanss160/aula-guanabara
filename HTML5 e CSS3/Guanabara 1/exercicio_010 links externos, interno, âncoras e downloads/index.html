Usando links externos

aqui estou usando uma palavra ou frase que ao clicar será usado um link que está vinculado nessa, usando o comando:
<a href="colocar o link aqui">palavra ou frase para o link</a>
segue exemplo:

acessando o site de melhor jogo(até o momento)
<a href="https://zenless.hoyoverse.com/pt-pt/main?utm_source=web&utm_medium=landingpage&utm_campaign=character">game zzz</a>

acessando o site de melhor jogo(até o momento) game zzz

Em seguida usarei a mesma frase, site e link, mas agora ao cliclar a página não se fechará porque abrirá outra e assim mantendo a original de onde o usuário saiu usando o comando:
<a href="colocar o link aqui" target="_blank" rel="external">palavra ou frase para o link</a>,
segue exemplo do novo comando:

acessando o site de melhor jogo(até o momento)<a href="https://zenless.hoyoverse.com/pt-pt/main?utm_source=web&utm_medium=landingpage&utm_campaign=character" target="_blank" rel="external">game zzz</a>

acessando o site de melhor jogo(até o momento) game zzz

Usando links internos:

Esta é a primeira página, caso queira ir para a segunda terá duas opções de link, o primeiro link usará esta aba para redirecionamento e o segundo manterá a aba original e abrirá outra. Estou usando o seguinte comando para a primeira opção:
<a href="nome da pagina ou colocar o link aqui">palavra ou frase para o link</a>
segue exemplo:

O primeiro link: <a href="pagina2.html">clique aqui</a>
O primeiro link: clique aqui

A segunda opção usarei o seguinte comando:
<a href="nome da pagina ou colocar o link aqui" target="_blank" rel="next">palavra ou frase para o link</a>
segue exemplo:

O segundo link: <a href="pagina2.html" target="_blank" rel="next">clique aqui</a>
O segundo link: clique aqui

Exemplo de uma terceira página em uma subpasta usando o comando:
<a href="nome da subpasta/nome da pagina ou link" target="_self" rel="next">palavra ou frase para o link</a>
segue exemplo:
Ir para a terceira página: <a href="teste_pagina/pagina3.html" target="_self" rel="next">clique aqui</a>
Ir para a terceira página: clique aqui,
neste caso você abrirá uma outra aba então lembre-se que os comandos da segunda página também são válidos, mas lembre de fazer a alteração necessária em (href="") porque o caminho é uma subpasta.

Links para downloads

Usarei o seguinte comando para download de um arquivo pdf e uma imagem, mas antes do link terá a imagem do que será baixado, lembrando que o arquivo está em uma subpasta:
<a href="nome da pasta/nome do arquivo.(pdf ou jpg)" download="nome do arquivo.(pdf ou jpg)" type="application/pdf ou image/jpeg">palavra ou frase para o link</a>,
segue exemplo:


  1. <a href="downloads/Golden Sign (feat. HOYO-MiX) TRADUÇÃO - Sãn-Z (Impressão).pdf" download="Golden Sign (feat. HOYO-MiX) TRADUÇÃO - Sãn-Z (Impressão).pdf" type="application/pdf">Baixar tradução da música: Golden Sing</a>
    Baixar tradução da música: Golden Sing


  2. <a href="nome da pasta/nome do arquivo.(pdf ou jpg)" download="nome do arquivo.(pdf ou jpg)" type="application/pdf ou image/jpeg">palavra ou frase para o link</a>
    Baixar imagem da: Astra Yao
