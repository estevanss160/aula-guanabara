Outras formatações

Código-fonte

esta frase não contem um espaçamento uniforme

esta frase contem um espaçamento uniforme usando o seguinte comando:
<code>texto para espaçamento uniforme</code>

usando apenas a formatação <code> não
é possível demonstrar o aninhamento/endentação conforme exemplo:
num = in(input("Digite um número: ")) if(num % 2 == 0): print(f"O número {num} é par") else: print(f"O número {num} é ímpar") print("Fim do programa") 
    
mas usando o comando <code> para manter o código uniforme e o comando <pre> antes de <code> 
é possível demonstrar o aninhamento/endentação conforme exemplo: 

num = in(input("Digite um número: "))
if(num % 2 == 0):
    print(f"O número {num} é par")
else:
    print(f"O número {num} é ímpar")
print("Fim do programa")
o comando "code" não é obrigatório para manter o aninhamento

Citações simples:

Para fazer citações simples basta usar o comando <q> que cria aspas duplas por exemplo:
De acordo com a frase: zzz é melhor kkkk
<q>texto para citação</q>

Citações completas:

Agora usarei o comando <blockquote cite="link da fonte"> para colocar a seguinte frase como uma citação fora do comando <p> e também poderia usar apenas o comando <blockquote> segue exemplo:

As melhores frazes de zzz são: as de duplo sentido kkkk 😏 segue o link:

Abreviações:

Estas abreviações por exemplo: HTML e CSS podem ser acompanhadas com seus significados se usar o comando
<abbr title="significado da abreviação">ABREVIAÇÃO</abbr> por exemplo:
<abbr title="HyperText Markup Language">HTML</abbr> e aplicando esse comando você poderá passar o mause na palavra sublinhada para verificar seu significado, segue exemplo:
HTML

Texto invertido usando o comando <bdo dir="rlt ou ltr">texto que será invertido</bdo>. Segue exemplo:
texto invertido
