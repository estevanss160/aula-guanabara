Trabalhando com listas:

Listas ordenadas(A ordem importa)

usando o comando <ol> e <li> posso criar uma lista numérica por exemplo: 

    <ol>
        <li>acordar</li>
        <li>desligar despertador</li>
        <li>levantar</li>
        <li>ir ao banheiro</li>
    </ol>
  1. acordar
  2. desligar despertador
  3. levantar
  4. ir ao banheiro
também posso colocar uma configuração específica para ordenar minha lista usando o comando
<ol type="1, A, a, I ou i"> seguido do comando <li> por exemplo: 
    <ol type="A">
        <li>acordar</li>
        <li>desligar despertador</li>
        <li>levantar</li>
        <li>ir ao banheiro</li>
    </ol>
  1. acordar
  2. desligar despertador
  3. levantar
  4. ir ao banheiro

Listas não ordenadas:

para criar uma lista não ordenada(A ordem não importa) basta usar o comando <ul> seguido do comando <li> segue exemplo: 

        <ul>
            <li>estudar khan academy</li>
            <li>estudar Python</li>
            <li>estudar HTML, CSS e Javascript</li>
            <li>estudar AWS</li>
        </ul>
  • acordar
  • desligar despertador
  • levantar
  • ir ao banheiro
também posso colocar uma configuração específica para ordenar minha lista usando o comando
<ul type="disc, circle e square"> seguido do comando <li> por exemplo: 
    <ul type="circle">
        <li>acordar</li>
        <li>desligar despertador</li>
        <li>levantar</li>
        <li>ir ao banheiro</li>
    </ul>
  • acordar
  • desligar despertador
  • levantar
  • ir ao banheiro

Lista de definições

usando o comando <dl>, <dt> e <dd> posso fazer marcações, segue exemplo: 

    <dl>
        <dt>HTML<dt>
        <dd>Linguagem de marcação para a criação do conteudo de um site.<dd>
        <dt>CSS<dt>
        <dd>Linguagem de marcação para criação do design de um site.<dd>
        <dt>JavaScript<dt>
        <dd>Linguagem de programação para a criação de interatividade de um site.<dd>
    </dl>
HTML:
Linguagem de marcação para a criação do conteúdo de um site.
CSS:
Linguagem de marcação para criação do design de um site.
JavaScript:
Linguagem de programação para a criação de interatividade de um site.
