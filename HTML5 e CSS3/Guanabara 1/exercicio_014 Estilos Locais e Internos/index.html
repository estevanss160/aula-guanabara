Capítulo 1.1

Neste exercicio foi trocado a cor de fundo do site, cor das letras, tamanho e fonte usando estilos de css com os deguintes comandos: 

    <style>
        comando para trocar a cor de fundo, fonte das letras e tamanho, Mas esse comando deve ser aplicado no "head" 
        do html mais específicamente depois de "title" e dentro de "style"
        body{
            background-color: nome da cor em inglês;
            font-family: Arial, Helvetica, sans-sherif;
            font-size; 20px
        }
        O "20" representa o tamanho e o "px" representa a unidade de medida usado
    </style>

Caítulo 1.2

ainda dentro de "style" no "head" do html podemos formatar todos os <h1> ou <h2> e assim sucessivamente segue exemplo: 

    <style>
        comando para trocar a cor de todos os "h1 ou h2...h6", Mas esse comando deve ser aplicado no
        "head" do html mais especificamente depois de "title" e dentro de "style"
        h1 ou h2...h6{
            color: nome da cor em inglês;
        }
    </style>
