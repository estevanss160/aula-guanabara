Capítulo 1.1

Neste exercicio foi trocado a cor de fundo do site, cor das letras, tamanho e fonte usando estilos de css com os deguintes comandos:
O primeiro comando é para usar um arquivo ".css" que criei para trocar a cor de fundo, fonte das letras e outros detalhes e esse comando deve ser aplicado no "head" do html mais específicamente depois de "title" e servirá para todas as páginas 

    <link rel="stylesheet" href"nome do arquivo.css">
    pense nesse código como se fosse um "import" do java
    e lá no arquivo .css colocarei os códigos de <body{}>, <h1...h6{}>, <p> e muitos outros no seguinte formato:
    body{
        background-color: nome da cor em inglês;
        font-family: Arial, Helvetica, sans-sherif;
        font-size; 20px
    O "20" representa o tamanho e o "px" representa a unidade de medida usado
    }
    h1 ou h2...h6{
        color: nome da cor em inglês;
    }
    p{
        text-align: tipo de formatação em inglês;
    }
lembre-se que de acordo com a HIERARQUIA os comandos INLINE sempre são superiores, exemplo na página 2
