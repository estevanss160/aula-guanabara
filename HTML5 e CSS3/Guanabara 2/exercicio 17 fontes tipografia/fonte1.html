Trabalhando com fontes

neste "html" foi usado no "head" o comando "style" e dentro dele foi colocado o seguinte comando: 

    <style>
        body {
            font-family:'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
            font-size: 1em ;
        }

        h1 {
            font-family:'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
            font-weight: bolder;
            font-size: 3em;
            font-style: italic;
        }

        h2 {
            /* font-style -> font-weight -> font-size -> font-family */
            font: italic bolder 3em 'work sans', sans-serif;
        }
    </style>

Explicação

os comandos "font-family" e "font-size" serve para definir qual o tipo de fonte/letra será usado e o tamanho respectivamente
lembrando que o "font-size" trabalha com vários tipos de unidade de medidas sendo as seguintes até onde sei:

MEDIDAS ABSOLUTAS:
cm, mm, in, px, pt, pc

MEDIDAS RELATIVAS:
em, ex, rem, vm, vh, %
obs:16px geralmente = 1em

em "h1" foi colocado 4 comandos que juntos configuram a fonte e em "h2" foi colocado o mesmo comando, mas de uma forma diferente;

em "h1" o comando "font-family: ..." diz qual fonte/letra deve ser usado seguindo as regras da esquerda para a direita
o comando "font-weight: ..." faz a fonte/letra ficar em negrito ou mais fina
o comando "font-size: ...", usando uma unidade de medida já explicada, configura o tamanho da fonte/letra
o comando "font-style: ..." é a forma em que a letra deve ser apresentada e neste caso está em ITALICO e essa configuração é usada no css3, mas em html é
usado o comando: <i>texto para ficar em italico</i>

em "h2" o comando foi compactado, mas segue os mesmos principios do que foi aplicado em "h1", o comando segue uma ordem que é a seguinte:
font-style -> font-weight -> font-size -> font-family
escolher a forma da letra -> escolher o quão "grossa" ela será -> escolher o tamanho da letra -> e por último o tipo de letra
