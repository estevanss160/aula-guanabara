USANDO INLINE

Exemplos de cores 1

neste exemplo foi colocado um "fundo" azul e trocado a cor das letras usando o seguinte comando dentro de h2:
<h2 style="background-color: nome da cor em inglês para alterar a cor do fundo; color: para trocar a cor da letra;">texto para receber a alteração</h2>
agora segue exemplo do comando usando nomes:
<h2 style="background-color: blue; color: white;">Exemplo de cores 1</h2>

Exemplos de cores 2

aqui foi aplicado o mesmo exemplo em h2, mas chamando as cores de uma forma diferente, segue o comando em hexadecimal:
<h2 style="background-color: #0000ff; color: #ffffff;">Exemplo de cores 2</h2>

Exemplos de cores 3

neste exemplo foi usado a mesma estrutura dos anteriores, porém a forma de chamar a cor
é diferente, segue exemplo em decimal:
<h2 style="background-color: rgb(0, 0, 255); color: rgb(255, 255, 255);">Exemplo de cores 3</h2>

Exemplo de cores 4

este exemplo é usado o "style" e "color", mas a forma de chamar a cor é com "hsl"
que significa (HUE, STURATION, LIGHTNESS) ou em português (MATIZ, SATURAÇÃO, LUMINOSIDADE), segue exemplo do comando:
<h2 style="background-color: hsl(240, 100%, 50%); color: hsl(1, 0%, 100%);">Exemplo de cores 4</h2>

considere que para cada uma das configurações existe também a possibilidade de transparência
por exemplo: "hsl" ficaria "hsla", "rgb" ficaria "rgba" e assim por diante e
essa nova variável se encaixaria após as outras e vai de 0 à 1, segue exemplo:
hsl(240, 100%, 50%, 0.205) ou rgb(0, 0, 255, 0.258)
