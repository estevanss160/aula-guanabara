para configurar essas cores no formato de gradiente ou degradê
foi usado o seguinte comando em CSS no "head" e após o "title" da página: 

    * {
        height: auto;
    }
    body {
        background-image: linear-gradient(to right, #3198e2, #6d59c0, #b93590, #e33f5f, #fdd579);
        background-attachment: fixed;
    }
o comando "*", antes das chaves, significa que todos os elementos do html foi selecionado

o comando "height: auto;", dentro das chaves, define a altura de todos os elementos automaticamente e quando falo altura estou falando da posição na página.

o comando "body", antes das chaves, que foi colocado no head do html5 seleciona todos os elementos do "<body>" que está após o head É IMPORTANTE SABER A DIFERENÇA

o comando "background-image: linear-gradient(...)", dentro das chaves, é o que criará o gradiente ou degradê da página com as cores que você selecionar, mas se a intenção
não é criar um gradiente ou degradê então criar uma cor sólida é o seguinte comando: background-color: nome da cor em inglês;

o comando "to right", dentro de "linear-gradient", serve para especificar que o degradê ou gradiente comece da esquerda para direita e pode ser mudado para outras formas por exemplo
se trocar o "linear-gradient" por "radial-gradient(circle...)"

o comando "background-attachment: fixed;", dentro das chaves, controla a imagem ou gradiente de fundo e o comando fixed mantem o fundo em estado estacionário criando
um efeito paralaxe
