Site de exemplo CSS3

Explicações:

neste exemplo foi alterado a fonte dos textos, as cores de cada texto(h1, h2), o fundo do site e também o fundo dos textos tudo isso usando CSS3. Para utilizar o CSS3 nesta página foi criado um arquivo chamado "style.css" na mesma pasta onde encrontra-se o arquivo "html" do site, mas lembrando que poderia ser colocado em uma subpasta também. Esse arquivo "css" contém todas as configurações de estilo do site e para chamar esse arquivo use o seguinte comando dentro do "head" e abaixo de "title":
<link rel="stylesheet" href="style.css">

o comando "<link>" é uma tag no "HTML5" é o que chama recursos externos.

o comando (rel="..."), dentro de "<link>", serve para especificar qual é o tipo do arquivo e "stylesheet" fala qual é o tipo de arquivo que o "href" vai usar que nesse caso é uma folha de estilos ou uma folha de configuração para a pagina "html" que tiver com esse "link" no "head".

o comando (href="..."), dentro de "<link>", expecífica qual o caminho do arquivo externo e "style.css" indica que está no mesmo diretorio do html.

Em resumo, o comando diz ao navegador: "Vá buscar o arquivo chamado style.css e use-o como a folha de estilos para estilizar esta página".

primeiro ARIAL se por um acaso ela não funcionar pode ser chamada a próxima que nesse caso é HELVETICA e se mesmo assim nenhuma funcionar tem uma carta na manga que é o SANS-SERIF que usa a fonte padrão do sistema.
