Usando fontes externas. segue comandos usados no "head" deste "html":

    <style>
        @import url('https://fonts.googleapis.com/css2?family=Dynalight&display=swap');
        
        @import url('https://fonts.googleapis.com/css2?family=Dynalight&family=Petit+Formal+Script&display=swap');

        @import url('https://fonts.googleapis.com/css2?family=Borel&family=Dynalight&family=Petit+Formal+Script&display=swap');

        h1 {
            font-family: "Dynalight", cursive;
            font-weight: 400;
            font-style: normal;
        }

        h2 {
            font-family: "Petit Formal Script", cursive;
            font-weight: 400;
            font-style: normal;
        }

        p {
            font-family: "Borel", cursive;
            font-weight: 400;
            font-style: normal;
        }
    </style>

Explicações do "@import"

para este exemplo foi usado fontes que normalmente, em teoria, não estão instaladas nos navegadores e nos computadores
para fazer essas fontes serem usadas foi usado um "@import", que é obrigatório usar abaixo do style ou nas primeiras linha da programação, e nesse caso
foi usado 3 import e cada um tras uma font diferente e o "h1, h2 e p" estão usando uma fonte diferente, mas para montar um site real use no máximo 4
segue o link para acessar o Google Fonts: https://fonts.google.com ou clique aqui

lembrando que existem fontes que não permitem certas configurações como por exemplo italico ou negrito
