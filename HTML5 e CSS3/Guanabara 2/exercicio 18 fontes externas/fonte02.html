Explicacão do "@font-face {}":

Neste exemplo foi usado o comando "@font-face {}" dentro de "style" no "head" deste "html", ele serve como um "@import", mas neste caso ele está pegando algo especifico como uma fonte e é uma fonte que baixei. O comando funciona em três partes a primeira é o "font-family: ..." onde será colocado o apelido/nome dessa fonte para que você possa chamar com esse apelido/nome o segundo passo é dizer onde o "CSS" pode buscar essa fonte usando o seguinte comando: "src: url('...')", a terceira parte é o "format(...)" que informa qual é o tipo do arquivo (truetype ou opentype...)

obeserve que o "font-family: japa" tem mais de um "url" e o motivo disso é que existem vários tipos de "format(...)" por exemplo: 

        Tipos de format():
        - opentype (otf)
        - truetype (ttf)
        - embedded-opentype
        - truetype-att (Apple advanced typography)
        - svg
        */

o motivo de colocar os dois tipos de "format" é que se o navegador que for abrir esse arquivo "html" não suportar um então terá o outro, mas e se não suportar nunhum? isso é uma ótima pergunta.

lembre também quem é possivel usar o comando "@font-face" e o comando "@import" no mesmo arquivo CSS
