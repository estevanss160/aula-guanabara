só posso dizer que tudo foi uma alucinação e que usar "div > p" significa que p é a filha de div e é possivel esconder tudo usando
"div:hover > p"

toma seu codigo: 

    <style>
        body {
            font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
        }

        div > p {
            display: none;
        }

        div:hover > p {
            width: 300px;
            display: block;
        }

        div#tentativadiv{
            display: none;
            position: absolute;
            width: 98%;
            height: 95%;
            border: 1px solid #ccc;
            background-color: #fff;
            z-index: 10;
        }

        div#tentativadiv iframe {
            width: 100%;
            height: 100%;
            border: none;
        }

        p#tentativaid:hover + div#tentativadiv  {
            display: block;
        }
    </style>
